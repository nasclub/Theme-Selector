# Theme Selector
This HTML module enables logged in users to select different themes for Nasclub.

> This is not yet rolled out publically on Nasclub.

> http://www.nasclub.co/admin/editmodule/index/preset/44424482

### Current Themes

* Default (Red)
* Purple Cloud (Purple)
* Deep Sea (Blue)
* Forest Lover (Green)
* Burnt Sunset (Orange)
* Black Sky (Grey)
